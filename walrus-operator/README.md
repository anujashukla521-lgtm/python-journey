# Walrus Operator (`:=`) Practice

## 📌 Overview

This folder contains my practice programs on Python's **Walrus Operator (`:=`)**.

The goal of this practice was to understand how the walrus operator can simplify code by combining **assignment** and **evaluation** in a single expression. I focused on solving practical problems rather than using the operator unnecessarily.

---

## 📚 Topics Covered

- Walrus Operator (`:=`)
- Assignment Expressions
- Input Validation
- While Loops
- Conditional Statements
- String Manipulation
- List Traversal
- Menu-Driven Programs
- Basic Interview Problems

---

## 📂 Programs

| File | Description |
|------|-------------|
| `positive_number_validation.py` | Keep asking for a positive number until valid input is entered. |
| `even_or_odd.py` | Check whether a number is even or odd. |
| `string_length_check.py` | Check the length of a string. |
| `sum_until_zero.py` | Continuously take numbers and calculate the sum until `0` is entered. |
| `guess_the_number.py` | Simple number guessing program. |
| `password_checker.py` | Validate the password using the walrus operator. |
| `word_count.py` | Count the number of words in a sentence. |
| `large_numbers.py` | Print numbers greater than a given value. |
| `long_username.py` | Display usernames based on length conditions. |
| `email_domain.py` | Extract the domain from an email address. |
| `read_until_quit.py` | Read user input until `"quit"` is entered. |
| `bonus_menu_system.py` | Menu-driven program using the walrus operator. |
