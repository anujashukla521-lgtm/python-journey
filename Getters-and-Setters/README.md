# Getters and Setters in Python

## Overview
This folder contains practice programs demonstrating the use of getters and setters in Python using the `@property` and `@property.setter` decorators. These programs focus on encapsulation, data validation, and computed properties, which are important concepts in Object-Oriented Programming (OOP).

## Topics Covered
- Getters using `@property`
- Setters using `@property.setter`
- Encapsulation
- Data Validation
- Computed Properties
- OOP Best Practices

## Key Learnings
- Understood how getters and setters work in Python.
- Learned to use `@property` and `@property.setter`.
- Practiced encapsulation using private attributes.
- Implemented validation logic to protect object data.
- Created computed properties that derive values dynamically.
- Improved understanding of OOP design principles.

Successfully implemented multiple programs using getters and setters, strengthening my understanding of encapsulation, validation, and property decorators in Python.
