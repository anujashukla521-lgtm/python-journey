# Dictionary Mini Projects

This folder contains beginner-level mini projects built using Python dictionaries.  
These projects are focused on improving logical thinking, CRUD operations, user interaction, and problem-solving skills.

---

# Mini Projects Included

## 1. dictionary_crud_system.py
A menu-driven CRUD (Create, Read, Update, Delete) application using Python dictionaries.

### Features
- Add items
- Update existing items
- Delete items
- Search items
- Display dictionary contents
- Exit system

### Concepts Used
- Dictionaries
- Loops
- Functions
- Recursion
- Match-case statements
- Conditional logic
