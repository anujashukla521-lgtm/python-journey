# Python Super Keyword

## 📖 Overview

This folder contains practice programs demonstrating the use of the `super()` keyword in Python. The programs focus on how child classes can access parent class constructors and methods, reinforcing the concepts of inheritance, constructor chaining, and method overriding.

---

## 📚 Topics Covered

- Introduction to the `super()` keyword
- Calling parent class constructors using `super().__init__()`
- Calling parent class methods using `super()`
- Constructor execution order
- Single inheritance
- Method overriding with `super()`
- Code reusability through inheritance

## 🎯 Key Learnings

- Understood the purpose of the `super()` keyword.
- Learned how to call parent class constructors using `super().__init__()`.
- Learned how to invoke parent class methods using `super()`.
- Understood the constructor execution order in single inheritance.
- Learned that if a child class defines its own constructor, the parent constructor is not called automatically.
