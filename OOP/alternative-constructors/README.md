# Alternative Constructors in Python

## 📌 Overview

This folder contains practice programs demonstrating **Alternative Constructors** in Python using the `@classmethod` decorator.

An alternative constructor provides an additional way to create objects when data is available in different formats such as strings, CSV data, or dictionaries. It keeps object creation clean, reusable, and easy to maintain.

---

## 📚 Topics Covered

* Class Methods
* Alternative Constructors
* Creating Objects from Strings
* Creating Objects from CSV Data
* Creating Objects from Dictionaries
* Multiple Constructors
* Object Initialization using `cls()`
