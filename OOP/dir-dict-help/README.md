# dir-dict-help

## 📌 Overview

This folder contains practice programs focused on Python's built-in introspection tools: `dir()`, `__dict__`, and `help()`. These features are useful for exploring objects, understanding available methods and attributes, inspecting object state, and reading built-in documentation.

---

## 📚 Topics Covered

* `dir()` function
* `__dict__` attribute
* `help()` function
* Methods vs Attributes
* Object Introspection
* Dynamic Attribute Addition

---

## 📂 File Structure

| File Name                           | Description                                                                          |
| ----------------------------------- | ------------------------------------------------------------------------------------ |
| `01_student_introspection.py`       | Uses `dir()`, `__dict__`, and `help()` with a custom `Student` class.                |
| `02_dynamic_attributes.py`          | Demonstrates adding attributes dynamically and observing changes in `__dict__`.      |
| `03_calculator_introspection.py`    | Explores custom class methods using `dir()` and `help()`.                            |
| `04_builtin_objects_exploration.py` | Compares built-in Python objects using `dir()` and examines their available methods. |

---

## 🎯 Key Learnings

* Learned how to inspect Python objects using `dir()`.
* Understood the purpose of the `__dict__` attribute for user-defined objects.
* Used `help()` to explore Python documentation for classes and functions.

* Learned how Python automatically passes `self` to instance methods.
* Explored dynamic attribute creation in Python objects.
* Improved understanding of Python introspection and debugging techniques.
