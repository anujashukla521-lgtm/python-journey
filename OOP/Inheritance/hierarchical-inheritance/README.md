# Hierarchical Inheritance in Python

This folder contains my practice programs on **Hierarchical Inheritance** in Python as part of my Object-Oriented Programming (OOP) learning journey.

## 📖 What is Hierarchical Inheritance?

Hierarchical Inheritance is a type of inheritance where **multiple child classes inherit from a single parent class**. It allows different classes to share common properties and methods while implementing their own unique functionalities.
