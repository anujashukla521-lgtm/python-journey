# Multilevel Inheritance in Python

## 📌 Overview

This folder contains programs demonstrating **Multilevel Inheritance** in Python. The implementations focus on understanding how inheritance works across multiple levels of a class hierarchy, how constructors are chained using `super()`, and how child classes inherit attributes and methods from their ancestors.

The examples are designed to strengthen object-oriented programming (OOP) concepts and prepare for more advanced inheritance topics such as Multiple Inheritance and Method Resolution Order (MRO).

---

## 🎯 Objectives

* Understand the concept of Multilevel Inheritance.
* Learn constructor chaining using `super().__init__()`.
* Access inherited attributes and methods across multiple inheritance levels.
* Implement method overriding in derived classes.
* Build a strong foundation for advanced OOP concepts.

---

## 📚 Topics Covered

* Multilevel Inheritance
* Constructor Chaining
* `super()` Function
* Method Overriding
* Attribute Inheritance
* Inheritance Hierarchy
* Parent and Child Class Relationships
