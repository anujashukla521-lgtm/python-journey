# Multiple Inheritance in Python

This folder contains Python programs demonstrating **Multiple Inheritance**, **Method Resolution Order (MRO)**, **Hybrid Inheritance**, and the use of the **super()** function. These examples help in understanding how Python resolves methods and constructors when multiple parent classes are involved.

---

## 📚 Topics Covered

- Multiple Inheritance
- Hybrid Inheritance
- Hierarchical Inheritance
- Method Resolution Order (MRO)
- `super()` Function
- Constructor Chaining
- Python's C3 Linearization Algorithm
- Real-world OOP Examples

---

## 📂 Programs Included

### 🔐 login_system.py
A login system demonstrating multiple inheritance by combining different user-related functionalities into a single class.

### 📱 smart_device_management.py
A smart device management system showing how multiple parent classes can provide different capabilities to a single device.

### 🔄 super_mro_demo.py
A demonstration of the `super()` function and Python's Method Resolution Order (MRO), explaining how constructors and methods are resolved in multiple inheritance.

### 👨‍🏫 teaching_assistant_system.py
A real-world example where a `TeachingAssistant` inherits features from both `Employee` and `Student`, illustrating multiple inheritance and constructor chaining.
