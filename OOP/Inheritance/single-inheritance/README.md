# Single Inheritance in Python

## 📌 Overview

This folder contains programs demonstrating the implementation of **Single Inheritance** in Python. These examples focus on understanding how a child class inherits attributes and methods from a single parent class, promoting code reusability and modular object-oriented design.

The programs cover real-world scenarios to strengthen the fundamentals of inheritance before progressing to advanced inheritance concepts.

---

## 🎯 Objectives

- Understand the concept of Single Inheritance.
- Learn how child classes inherit properties from parent classes.
- Practice constructor inheritance using `super()`.
- Implement method overriding.
- Build a strong foundation for Object-Oriented Programming (OOP).

---

## 📚 Topics Covered

- Single Inheritance
- Parent and Child Classes
- Constructor Inheritance
- `super()` Function
- Method Overriding
- Attribute Inheritance
- Code Reusability
