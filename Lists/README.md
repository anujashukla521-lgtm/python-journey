## Python Lists Practice
This folder contains beginner-friendly Python programs based on Lists.
The programs focus on list operations, traversal, searching, sorting, frequency counting, and basic logic-building concepts.

## Topics Covered

- List Creation
- List Traversal
- Conditional Checking
- Merging Lists
- Positive & Negative Number Separation
- Manual Sorting
- Frequency Counting
- Finding Second Largest Element

## Purpose
The purpose of these programs is to strengthen understanding of Python lists and improve problem-solving skills through hands-on practice.

## Key Learnings

- Using loops with lists
- Applying conditions on list elements
- Swapping elements manually
- Counting occurrences of elements
- Building logic without relying completely on built-in methods
