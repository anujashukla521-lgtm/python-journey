# Day 54 - Clear The Clutter

## Overview
A Python script that renames all PNG files inside a folder sequentially.

## Concepts Used
- os module
- os.listdir()
- os.rename()
- Loops
- Conditional statements
- File handling basics

## How It Works
1. Reads all files from the cluttered_folder.
2. Filters PNG files.
3. Renames them as 1.png, 2.png, 3.png, etc.

## Learning Outcome
Learned how to interact with directories and rename files using the os module.
