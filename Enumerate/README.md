## Enumerate in Python

This folder contains practice programs based on Python's enumerate() function. These programs help in understanding how to access both index positions and values while looping through lists, strings, and other iterable objects.

## Topics Covered
- Using enumerate() with lists
- Accessing index and value together
- Custom starting index
- String traversal with indexes
- Duplicate detection
- Numbered menus and lists

## Purpose

The purpose of these programs is to practice:
- Looping techniques
- Index handling
- Cleaner and more Pythonic iteration
- Real-world usage of enumerate()
  
## Key Learning
enumerate() makes loops more readable and efficient compared to manually managing indexes using range(len()).
# Example:
for index, value in enumerate(items):
    print(index, value)
