# Import Modules in Python

## 📌 Purpose

This folder contains practice programs created while learning Python modules and imports. The goal of these exercises is to understand how to organize code into multiple files, improve reusability, and build modular applications.

---

## 📚 Topics Covered

* Creating custom modules
* Importing modules using `import`
* Importing specific functions using `from module import function`
* Function reusability
* Code organization
* Modular programming
* Input validation
* Utility functions

---

## 📂 Programs Included

### 1. Calculator Module

* Basic arithmetic operations
* Importing functions from a custom module

### 2. Greeting Module

* Function imports
* Calling functions from another file

### 3. Student Result System

* Percentage calculation
* Pass/Fail determination
* Utility function usage

### 4. Password Checker Modular

* Password length validation
* Digit validation
* Validation functions in separate modules

---

## 🛠 Technologies Used

* Python 3

---

## 🎯 Key Learnings

* How Python imports work
* Difference between `import module` and `from module import function`
* Organizing code into multiple files
* Building reusable functions
* Writing cleaner and more maintainable programs
