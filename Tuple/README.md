## Tuple Programs in Python
This folder contains Python programs based on Tuples, one of Python’s built-in immutable data types.
The programs cover tuple creation, indexing, slicing, built-in methods, logical operations, and problem-solving concepts.

## Topics Covered

- Tuple creation and access
- Tuple slicing
- Tuple methods (count(), index())
- Tuple concatenation and repetition
- Membership checking
- Nested tuples
- Tuple unpacking
- Palindrome checking
- Common elements between tuples
- Frequency counting
- Even number extraction
- Maximum and minimum element finding

## Purpose
The purpose of this folder is to strengthen understanding of:
- Immutable data structures
- Iteration through tuples
- Logical problem-solving
- Python sequence operations
- Tuple-based interview practice questions

## Technologies Used
Python 3

## Key Learnings

- Difference between lists and tuples
- Immutable nature of tuples
- Efficient iteration techniques
- Using slicing for reverse operations
- Handling duplicates and comparisons
- Writing logic-based programs using tuples
