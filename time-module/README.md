# ⏰ Python Time Module Practice

This folder contains my hands-on practice programs for Python's built-in `time` module. The goal was to understand how time-related functions work by implementing small utilities, timers, clocks, and mini projects instead of only learning the theory.

---

## 📚 Topics Covered

- Current Date & Time
- Time Formatting (`strftime`)
- `time.sleep()`
- `time.time()`
- Execution Time Measurement
- Stopwatch
- Countdown Timer
- Alarm Clock
- Digital Clock
- World Clock (Local vs UTC)
- Typing Speed Calculator
- Quiz Timer
- Study Session Tracker

---

## 📂 Programs

| File | Description |
|------|-------------|
| `current_time.py` | Display the current system time |
| `time_formatter.py` | Format date and time using `strftime()` |
| `greeting_based_on_time.py` | Display greetings based on the current time |
| `loading_delay.py` | Simulate a loading animation using `sleep()` |
| `execution_time_calculator.py` | Measure program execution time |
| `stopwatch.py` | Simple stopwatch implementation |
| `countdown_timer.py` | Countdown timer using hours, minutes, and seconds |
| `digital_clock.py` | Display the current time continuously |
| `live_digital_clock.py` | Live updating digital clock |
| `alarm_clock.py` | Alarm that triggers at a specified time |
| `world_clock.py` | Compare Local Time and UTC Time |
| `typing_speed_timer.py` | Calculate typing speed (WPM) |
| `quiz_timer.py` | Quiz program with answer timing |
| `study_timer_countdown.py` | Study countdown timer |
| `study_session_tracker.py` | Track multiple study sessions and total study time |
