## Harry Potter KBC Game

A Harry Potter themed KBC-style quiz game built using Python.
Players answer multiple-choice questions to win virtual prize money while progressing through different levels of the game.

## Features
- Multiple-choice Harry Potter trivia questions
- Prize money system
- Safe point levels
- Correct and wrong answer handling
- Final winnings calculation
- Interactive command-line gameplay
  
## Concepts Used
- Lists
- Dictionaries
- Loops
- Conditional Statements
- User Input Handling
- Nested Data Structures
