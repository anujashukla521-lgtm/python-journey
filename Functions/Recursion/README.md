# Recursion in Python

This folder contains Python programs based on the concept of **recursion**.  
Recursion is a programming technique where a function calls itself to solve smaller parts of a problem.

## Topics Covered
- Recursive Functions
- Base Case and Recursive Case
- Factorial using Recursion
- Fibonacci Sequence using Recursion

## Programs Included

### 1. Factorial using Recursion
A Python program that calculates the factorial of a number using a recursive function and handles negative number input.

### 2. Fibonacci Sequence using Recursion
A Python program that generates the Fibonacci sequence using recursion and displays the sequence up to a user-specified range.

## Concepts Practiced
- Function calls
- Recursive thinking
- Return statements
- Problem breakdown
- Mathematical recursion logic

## Learning Outcome
Through these programs, I practiced how recursion works internally and how problems can be solved by reducing them into smaller subproblems.
