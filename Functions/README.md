## Functions in Python
This folder contains beginner-to-intermediate level Python programs focused on understanding and practicing functions.
The programs cover core concepts such as function creation, parameters, return values, conditional logic, loops inside functions, and basic problem-solving techniques.

## Topics Covered

- Function Definition
- Function Calling
- Parameters & Arguments
- Return Statements
- Multiple Return Values
- Conditional Logic in Functions
- Loops Inside Functions
- String Handling with Functions
- Modular Programming Basics
