# Map, Filter & Reduce in Python

## Overview

This folder contains practice programs demonstrating the use of Python's functional programming utilities: **map()**, **filter()**, and **reduce()**. These functions help perform transformations, filtering, and aggregations on iterable data in a concise and efficient manner, often in combination with lambda functions.

---

## Topics Covered

### map()

Applies a function to every element of an iterable and returns a map object.

### filter()

Filters elements from an iterable based on a condition and returns only the matching elements.

### reduce()

Applies a function cumulatively to the elements of an iterable and reduces them to a single value.

### Lambda Functions

Anonymous functions commonly used with map(), filter(), and reduce() for concise implementations.
