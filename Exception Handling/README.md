# Exception Handling in Python

##  Overview

This folder contains my practice programs for learning **exception handling in Python**.
It includes small programs and mini exercises that demonstrate how to handle runtime errors and prevent program crashes.

---

##  Topics Covered

* try and except blocks
* Handling `ValueError`
* Handling `ZeroDivisionError`
* Handling `IndexError`
* Handling `KeyError`
* Input validation
* Basic error prevention techniques

---

##  Purpose

The goal of this folder is to:

* Understand how errors occur in Python programs
* Learn how to handle errors gracefully
* Improve program reliability
* Practice writing safe user input programs
* Build a strong foundation for real-world coding

---

##  Key Learnings

* Difference between syntax errors and runtime errors
* When to use specific exceptions instead of generic ones
* Combining logic (if-else) with exception handling
* Writing user-friendly error messages
