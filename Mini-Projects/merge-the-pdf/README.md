# Merge the PDF

A simple Python mini project that demonstrates how to merge multiple PDF files into a single PDF using the **pypdf** library.

---

## 📌 Overview

This project combines multiple PDF files into a single output PDF using Python. It introduces working with third-party libraries, file handling, and PDF manipulation.

---

## 🚀 Features

- Merge multiple PDF files into one PDF
- Generate a single output PDF
- Simple and beginner-friendly implementation
- Uses the `pypdf` library

---

## 🛠️ Technologies Used

- Python 3
- pypdf

---

## 📂 Project Structure

```text
merge-the-pdf/
│── merge-pdf.py
│── example.pdf
│── hello-world.pdf
│── out-basic.pdf
│── README.md
```

---

## ▶️ How to Run

### 1. Install the required package

```bash
pip install pypdf
```

### 2. Run the program

```bash
python merge-pdf.py
```

After successful execution, a new file named **out-basic.pdf** will be created in the project folder.
