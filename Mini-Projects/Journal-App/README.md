# Journal App

## Overview

A simple command-line Journal Application built with Python that allows users to write, view, and search journal entries stored in a text file.

## Features

* Add new journal entries
* View all saved journal entries
* Search journal entries using keywords
* Input validation for empty entries
* Exception handling for missing files
* Menu-driven interface

## Concepts Used

* Functions
* File Handling (`read` and `append` modes)
* Exception Handling (`try-except`)
* String Methods (`strip()`, `lower()`)
* Loops
* Conditional Statements
* Match-Case
* Input Validation

1. Clone the repository.
2. Navigate to the project folder.
3. Run the program:

