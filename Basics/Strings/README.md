## String Programs in Python
This folder contains beginner-friendly Python programs based on String Operations and Methods.
These programs helped me practice important string concepts like slicing, indexing, formatting, validation, and built-in string functions.

## Topics Covered

- String Indexing
- String Slicing
- Reverse Strings
- Uppercase & Lowercase Conversion
- strip() Method
- replace() Method
- count() Method
- startswith() and endswith()
- capitalize() and title()
- Finding String Length
- Accessing Middle Characters
- Basic Email Validation

## Purpose
The purpose of these programs is to strengthen Python string handling concepts through small practical examples.

## Technologies Used
Python 3

## Key Learnings

- How to manipulate strings using slicing
- Using built-in string methods effectively
- Understanding indexing and traversal
- Performing basic validations and formatting
