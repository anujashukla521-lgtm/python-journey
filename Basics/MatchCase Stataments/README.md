## Match-Case Programs in Python

This folder contains beginner-friendly Python programs based on match-case statements.
The programs demonstrate menu-driven logic, conditional handling, arithmetic operations, string handling, and user input processing.

## Programs Included
- Day Finder
- Simple Calculator
- Traffic Light System
- Month Name Finder
- Vowel Checker
- Grade Message
- Fruit Price Checker
- Language Greeting
- ATM Menu System
- Even or Odd Checker
  
## Concepts Practiced
- match-case statements
- Conditional logic
- User input handling
- String methods
- Arithmetic and modulo operations
  
## Technologies Used
Python 3
